# 🌤️ Weather API

Aplicação web simples que busca informações de clima ao digitar o nome de uma cidade.

## 🔎 Como funciona

O usuário digita o nome de uma cidade e a aplicação consulta uma API de clima para retornar informações atualizadas.

Exemplo:
Curitiba
São Paulo
London

## 📊 Informações exibidas

- temperatura
- condição do tempo
- nome da cidade

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Fetch API

## ⚙️ Como usar

1. Abra o arquivo `index.html`
2. Digite o nome de uma cidade
3. Veja as informações climáticas retornadas pela API

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- consumo de APIs
- manipulação de JSON
- requisições assíncronas em JavaScript

## 📚 Aprendizados

Durante o desenvolvimento foram praticados:

- uso de `fetch()`
- tratamento de respostas da API
- manipulação do DOM
